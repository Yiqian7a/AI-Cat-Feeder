### 2.2 修改gpio映射

进入OPi库的路径修改

```bash
cd /usr/local/lib/
find . -name "OPi*"
```

进入OPi文件夹下，找到`pin_mappings.py`文件

将下面的映射字典替换原有的字典，注意只要修改`BOARD{ }`中的内容：

